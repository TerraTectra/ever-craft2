# Russian Localization Glossary

Core terms used consistently in RU localization:

- `Ascension` -> `Вознесение`
- `Mastery` -> `Мастерство`
- `Singularity` -> `Сингулярность`
- `Realm` -> `Мир`
- `Prime Realm` -> `Первичный мир`
- `Unified Realm` -> `Единый мир`
- `Realm of War` -> `Мир войны`
- `Realm of Magic` -> `Мир магии`
- `Realm of Earth` -> `Мир земли`
- `Expedition` -> `Экспедиция`
- `Artifact` -> `Артефакт`
- `Prestige` -> `Престиж`
- `Altar` -> `Алтарь`
- `Tier` -> `Тир`
- `Mana` -> `Мана`
- `Arcane Soul` -> `Тайная душа`
- `Soulforge` -> `Кузница душ`
- `Time Warp` -> `Перемотка времени`

Style rules:
- Keep game-system nouns capitalized only when they are section names.
- Keep `T1/T2/...` notation unchanged.
- Prefer concise UI wording over literal long translations.
